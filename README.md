# Instalation 
```
 docker compose up
 composer install
 php artisan migrate
 php artisan db:seed
```
## Design feature

- Реализация валидации форм
- Реализация пагинации
- Реализация поля поиска
- Реализация генерирования PDF-документа 
- Реализация SoftDelete 
- Реализация CRUD
